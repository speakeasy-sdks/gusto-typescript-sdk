# CompensationPaymentUnit

The unit accompanying the compensation rate. If the employee is an owner, rate should be 'Paycheck'.

| ---------- | ---------- |
| `Hour`     | Hour       |
| `Week`     | Week       |

| `Year`     | Year       |
